    Studio Surface 标准切换台将 Studio 所有常用功能都集成到硬件按钮和推子上，通过直观易懂和常规化的面板，以及集成到切换台上的LED屏幕，无需学习就能上手操作。触觉控制能让你以最快速度访问和控制关键功能，其中包括完整的音频水平、混音、切换、分组、联动等，可编程式功能按钮与推子结合面板监视器更可一一应对当前 Studio 版本所有常用功能和即将推出的新功能。
