    支持 4K 画面裁切虚拟机位

    支持任意机位创建虚拟机位。
    当推流至 1080 全高清或 720 高清时最好使用 4K 视频源。
    支持虚拟机位摇拍、切换和自动化预置。

    冗余推流

    冗余推流让你将直播视频流按不同码率经 3G／4G、Wi-Fi、有线网络和其它操作系统能识别到的不同链路推送至指定直播平台，带给你更稳定的直播体验。
    你还可以设置一个备用推流连接，当你主网络太慢或连接失败时可以轻松切换。尽可能保证播出安全稳定。

    了解更多关于冗余推流

    *此功能暂时只对 Livestream 平台直播开放
